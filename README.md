# k3-addons: Additional multi-backend functionality for Keras 3.
![Logo](.assets/k3-addons.png)

# Includes:
- Layers
    - Pooling:
        - `AdaptiveAveragePooling1D`
        - `AdaptiveMaxPooling1D`
        - `AdaptiveAveragePooling2D`
        - `AdaptiveMaxPooling2D`
    - Attention:
        - `DoubleAttention`
        - `AFTFull`
        - `ChannelAttention2D`
        - `SpatialAttention2D`
        - `ECAAttention`
        - `ExternalAttention`
        - `ResidualAttention`
